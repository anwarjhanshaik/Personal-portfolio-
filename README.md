# 🌐 Personal Developer Portfolio

A sleek, responsive, and lightweight personal portfolio website built with semantic HTML5 and standard CSS3. This project serves as a centralized digital hub to showcase my front-end development capabilities, technical skills, production-ready projects, and professional background.

Live Demo: [https://anwarjhanshaik.github.io/Personal_Portfolio/]

---

## ✨ Key Features

* **🚀 Dynamic Hero Section**: A clean, high-impact introductory fold featuring a professional summary and immediate call-to-action hooks.
* **📂 Selected Projects Showcase**: A structured layout highlighting production work, including deep dives into individual repository technicalities (such as custom media/print queries and zero-dependency structures).
* **📇 Integrated Social Hub**: Direct, accessible hyperlinking to my active engineering and professional profiles across external platforms.
* **📬 Accessible Contact Section**: A streamlined contact region making it frictionless for recruiters, clients, or collaborators to get in touch.
* **📱 Fully Fluid Layout**: Hand-coded mobile-first responsive architecture that adjusts seamlessly from tiny mobile screens to large desktop monitors without any layout shift.

---

## 🛠️ Built With

* **HTML5**: Structured with strict semantic components (`<header>`, `<section>`, `<main>`, `<footer>`) to optimize SEO and accessibility (a11y).
* **CSS3**: Layout styled natively using Flexbox, CSS Grid layout systems, and native fluid typography.

## 🚀 Installation & Local Development

To run this portfolio locally on your machine for customization:

### 1. Clone the Repository
```bash
git clone https://github.com
```

### 2. Launch Locally
Simply open the `index.html` file directly in any modern web browser, or use a local development server extension like "Live Server" in VS Code to see live changes as you code.
